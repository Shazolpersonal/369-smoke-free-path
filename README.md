**বিসমিল্লাহির রাহমানির রাহীম**

**৩৬৯ ধোঁয়া-মুক্ত পথ** হলো একটি মোবাইল অ্যাপ্লিকেশন যা আপনাকে ৪১ দিনের একটি বিশেষ যাত্রার মাধ্যমে ধূমপান ছাড়তে সাহায্য করবে। এই অ্যাপটি নিকোলা টেসলার বিখ্যাত **"৩৬৯ ম্যানিফেস্টেশন মেথড"**, পজিটিভ অ্যাফারমেশন (ইতিবাচক বাক্য) এবং ইসলামিক মূল্যবোধের সমন্বয়ে তৈরি করা হয়েছে, যাতে ব্যবহারকারীরা আধ্যাত্মিক এবং মানসিকভাবে শক্তিশালী হয়ে ধূমপানের ক্ষতিকর অভ্যাস ত্যাগ করতে পারেন।

## 🎯 প্রজেক্টের উদ্দেশ্য

ধূমপান ছাড়া একটি কঠিন মানসিক যুদ্ধ। এই অ্যাপটির মূল উদ্দেশ্য হলো এই যুদ্ধকে একটি নিয়মতান্ত্রিক এবং আধ্যাত্মিক যাত্রায় রূপান্তর করা। প্রতিদিনের তিনটি নির্দিষ্ট সময়ে (সকাল, দুপুর এবং রাত) ইতিবাচক বাক্য লিখে মনকে নতুনভাবে প্রোগ্রাম করার মাধ্যমে এই অ্যাপ আপনাকে সাহায্য করবে।

## ✨ মূল ফিচারসমূহ

- **৩৬৯ মেথড প্রয়োগ:** প্রতিদিন সকালে ৩ বার, দুপুরে ৬ বার এবং রাতে ৯ বার ইতিবাচক অ্যাফারমেশন লেখার রুটিন।
- **৪১ দিনের জার্নি:** ধূমপান ছাড়ার সম্পূর্ণ প্রক্রিয়াটিকে ৪১ দিনের একটি ট্র্যাকিং সিস্টেমে ভাগ করা হয়েছে।
- **দ্বিভাষিক সাপোর্ট:** বাংলা এবং ইংরেজি—উভয় ভাষাতেই অ্যাপটি ব্যবহার করা যাবে।
- **অফলাইন ফার্স্ট:** ইন্টারনেট সংযোগ ছাড়াই অ্যাপটি সম্পূর্ণভাবে কাজ করে। ডেটা লোকালি সেভ করা হয়।
- **প্রোগ্রেস ট্র্যাকিং এবং স্ট্রিক:** আপনার প্রতিদিনের উন্নতি দেখতে পারবেন এবং 'স্ট্রিক' বা ধারাবাহিকতা বজায় রাখার মাধ্যমে উৎসাহিত হতে পারবেন।
- **স্মার্ট নোটিফিকেশন:** সেশনগুলো আনলক হওয়ার সময় এবং রুটিন অনুযায়ী আপনাকে মনে করিয়ে দেওয়ার জন্য নোটিফিকেশন সিস্টেম।
- **সুন্দর এবং শান্তিময় ইউজার ইন্টারফেস:** ব্যবহারকারীর মনে প্রশান্তি আনার জন্য ইসলামিক জিওমেট্রিক প্যাটার্ন এবং কালার প্যালেট ব্যবহার করে ডিজাইন করা হয়েছে।

## 🚀 কিভাবে প্রজেক্টটি লোকাল মেশিনে রান করবেন?

এই প্রজেক্টটি রান করার জন্য আপনার মেশিনে `Node.js` ইনস্টল থাকতে হবে। প্যাকেজ ম্যানেজার হিসেবে **`pnpm`** ব্যবহার করার পরামর্শ দেওয়া হচ্ছে।

### ধাপ ১: রিপোজিটরি ক্লোন করুন
```bash
git clone https://github.com/Shazolpersonal/369-smoke-free-path.git
cd 369-smoke-free-path
```

### ধাপ ২: ডিপেন্ডেন্সি ইনস্টল করুন
> **নোট:** প্যাকেজ ইনস্টল বা স্ক্রিপ্ট রান করার জন্য `npm` বা `yarn` এর পরিবর্তে **`pnpm`** ব্যবহার করুন।

```bash
npm install -g pnpm  # যদি pnpm ইনস্টল করা না থাকে
pnpm install
```

### ধাপ ৩: অ্যাপ চালু করুন
```bash
pnpm start
```
এটি Expo-এর ডেভেলপমেন্ট সার্ভার চালু করবে। এরপর:
- আপনার ফোনে **Expo Go** অ্যাপটি ইনস্টল করে টার্মিনালে দেখানো QR কোডটি স্ক্যান করুন।
- অথবা, Android এমুলেটর (`a` প্রেস করে) বা iOS সিমুলেটর (`i` প্রেস করে) ব্যবহার করে অ্যাপটি রান করুন।
